식당에서 음식을 주문하는 다양한 사례를 수집 중이다.
너의 할 일은 다양한 주문 문장을 만들어 주는 것이다.

그런데 다음 조건을 충족시켜주면 좋겠어.
1. 음식명은 아래 "음식명 리스트"에서 최소 1개에서 최대 3개까지 샘플링해서 포함시켜줘.
2. 만약 1번에서 샘플링한 음식명 자체에 옵션이나 수량이 포함되어 있다면 분리해서 사용해줘.
3. 만약 1번에서 샘플링한 음식명에 다수의 음식이 혼합되어 있다면 대표 음식만 남겨줘.
4. 만약 음식명이 아닌 것 같으면 제외시켜줘.
5. 음식에는 옵션과 수량이 동반될 수 있어.
6. 맨 아래 든 예시처럼 주문 문장 마다 분석결과를 음식명, 옵션명, 수량 형식으로 생성되게 해줘.

예시를 들어 줄게;
예시1 주문 문장: 오늘은 비가오니깐 이거 먹자. 삼선짬뽕 곱배기하구요, 사천 탕수육 중짜 한그릇 주세요.
- 분석 결과 0: 음식명:삼선짬뽕,옵션:곱배기
- 분석 결과 1: 음식명:사천 탕수육,옵션:중짜,수량:한그릇

예시2 주문 문장: 아이스아메리카노 톨사이즈 한잔 하고요. 딸기스무디 한잔 주세요. 또, 콜드브루라떼 한잔입니다.
- 분석 결과 0: 음식명:아이스아메리카노,옵션:톨사이즈,수량:한잔
- 분석 결과 1: 음식명:딸기스무디,수량:한잔
- 분석 결과 2: 음식명:콜드브루라떼, 수량: 한잔

예시3 주문 문장: 참이슬 한병, 코카콜라 1.5리터 한병, 테슬라 한병이요.
- 분석 결과 0: 음식명:참이슬,수량:한병
- 분석 결과 1: 음식명:코카콜라,옵션:1.5리터,수량:한병
- 분석 결과 2: 음식명:테슬라,수량:한병

예시4 주문 문장: 꼬막무침 1인분하고요, 닭도리탕 중자 주세요. 그리고 소주도 한병 주세요.
- 분석 결과 0: 음식명:꼬막무침,수량:1인분
- 분석 결과 1: 음식명:닭도리탕,옵션:중자
- 분석 결과 2: 음식명:소주,수량:한병

예시5 주문 문장: 불고기버거세트 1개하고요 감자튀김 추가해주세요.
- 분석 결과 0: 음식명:불고기버거,수량:1개
- 분석 결과 1: 음식명:감자튀김

예시6 주문 문장: 불닭볶음면 1개랑 사리곰탕면 2개 주세요.
